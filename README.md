# VLKit:Vision and Learning Toolkit

## Installation
Install the latest version from GitHub:
```
pip install git+https://github.com/batchfy/vlkit.git@main
```

or install from source:
```
git clone https://github.com/batchfy/vlkit.git
cd vlkit
pip install -e ./
```
